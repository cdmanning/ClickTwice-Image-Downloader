# ![](documentation/ReadMeIcon.png) Click-Twice Image Downloader

A chrome extension for simplifying the task of downloading images from a website with a simple double-click.

## ✨ Features

* **Instant On/Off Toggle** 
* **Saves Seconds Per Day**
* **Reduces Carpal Tunnel**
  
## 🛠️ Installation & Usage

### 1. Installation

Install the "Click-Twice Image Downloader" extension directly from the [Chrome Web Store]() <--- This link doesn't actually link to anything as the Extension is currently pending approval.

### 2. Download Images

Download images with a simple double-click:

1. Find an image you wish to download.
2. Hover your cursor over the image.
3. Click on the image twice in rapid succession.
4. The download prompt will automatically appear.
